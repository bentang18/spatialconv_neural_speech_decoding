# Cross-Patient Neural Speech Decoding

**[Read the overview (PDF)](docs/physics_informed_architecture.pdf)**

Cross-patient speech decoding from intraoperative uECoG recordings using atlas-grounded cortical nodes and physics-informed spatial pooling. 16 cortical nodes at Brainnetome atlas positions map variable electrode arrays into a shared functional space. 182 learned parameters per patient.

Ben Tang | Cogan Lab, Duke University
