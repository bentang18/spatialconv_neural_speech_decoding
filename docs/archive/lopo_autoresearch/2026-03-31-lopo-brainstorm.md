## Diagnosis: The Core Bottleneck

Stage 1 training processes **one patient at a time per batch**. The backbone can exploit patient-specific feature distributions as shortcuts — patient-specific features correlate with labels within each patient but don't transfer. All 16 failed experiments kept this sequential training loop unchanged.

The backbone produces features where **patient identity dominates phoneme identity**. k-NN works slightly better than linear heads because it does instance-level matching rather than learning a decision boundary on this corrupted space.

### Rank 1: Mixed-Patient Batches in Stage 1

**Key insight**: Current training iterates patients sequentially — each gradient step contains only one patient. The backbone never sees mixed-patient batches and can learn patient-specific shortcuts.

**Implementation**: Sample 2 patients per gradient step, process through their respective read-ins, concatenate features at backbone input level (both are d=256), compute loss jointly, single backward pass.

### Rank 2: Nearest-Centroid / Prototype Eval

Compute per-class centroids (mean embedding per phoneme per position from train + weighted source embeddings). Classify by cosine similarity to nearest centroid. More robust than k=10 NN to outliers.

### Rank 3: Multi-Seed Ensemble

Run 3 seeds (42, 137, 256), average logits/softmax before decoding. Reduces init-dependent variance (~0.022 std across folds).

### Rank 5: Per-Patient Z-Score After Read-In

After read-in projects to d=256, apply per-batch z-score normalization (subtract mean, divide std). Forces backbone to see standardized inputs stripped of patient-specific amplitude/offset.

### Rank 7: SimpleShot (Transductive Eval)

Center embeddings (subtract mean of train embeddings) + L2 normalize + nearest centroid. Gold-standard few-shot baseline. Pure eval change, adds ~5 lines.

### Rank 10: Temporal Segment Voting

Split 20 BiGRU frames into thirds (0-6, 7-13, 14-19). Decode position 1 from first third, etc. Uses temporal locality of CVC production as inductive bias.
